📐 Understanding Bias

Simulate systematic error — see how bias shifts results, how CRM and EQA detect it, and how it combines with imprecision to affect patients.

Exercise 1
No bias — method is correct
Results cluster symmetrically around the true value. No systematic shift in either direction.
Bias = 0% CV = 2% TAE = 6%
Exercise 2
Positive bias — results too high
A calibration drift shifts all results upward. Bias is systematic — it does not cancel out over many measurements.
Bias = +4% CV = 2% TAE = 6%
Exercise 3
Bias passes — but TE fails
Bias is within the allowable limit. But combined with imprecision, total error exceeds TAE. Both must be evaluated together.
Bias = 2% CV = 3% TAE = 6%
Exercise 4
EQA bias — matrix effect?
EQA shows apparent bias. But the same method on native patient samples shows no bias. This is a matrix effect, not a real analytical problem.
EQA bias = +5% Patient bias = 0% CV = 2%
Exercise 5
Negative bias — results too low
Results are consistently below the true value. For some analytes, underestimation is clinically more dangerous than overestimation.
Bias = −3.5% CV = 1.5% TAE = 5%
