Visualisations In report ▼
Accuracy & limitations: Marginal mean plots show estimated group distributions assuming normality within each cell. The interaction plot displays cell means ± 1 SE — non-parallel lines suggest an interaction, but visual patterns can be misleading with small cell sizes or large SE. Two-way ANOVA requires normality within each cell, homogeneity of variance, and a balanced or near-balanced design. Severely unbalanced cells or small n per cell reduce reliability.
Factor A — Marginal Means
Each bell curve shows the data distribution for each group using its SD (wider curves = more variability within that group). The dashed line is the grand mean. Arrows above the curves show between-group spread (each group mean → grand mean, drives F numerator). Arrows below show within-group spread (±3 SD per group, drives F denominator). Colours match each group's curve.
Factor B — Marginal Means
Each bell curve shows the data distribution for each group using its SD (wider curves = more variability within that group). The dashed line is the grand mean. Arrows above show between-group spread (group mean → grand mean). Arrows below show within-group spread (±3 SD). Colours match each group's curve.
Interaction Plot
Cell means for every combination of Factor A and Factor B. Non-parallel lines indicate an interaction effect. Error bars = ±1 SE.
