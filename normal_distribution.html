Formulas used in this tool

Population (Normal distribution)

Use this when the population standard deviation σ is treated as known (or assumed known).

Standardization:
Z = (X − μ) / σ
Back-transform:
X = μ + Z · σ
Probability between two X-values:
P(a ≤ X ≤ b) = Φ(Z₂) − Φ(Z₁)

Sample (t distribution)

Use this when the population σ is unknown and we substitute the sample SD s. The distribution depends on degrees of freedom.

Standard error:
SE = s / √n
Standardization:
t = (X − μ) / (s / √n)
Degrees of freedom:
df = n − 1
Back-transform:
X = μ + t · (s / √n)
Probability between two limits:
P(a ≤ X ≤ b) = Fₜ(t₂) − Fₜ(t₁)

Here, Φ is the standard normal CDF and Fₜ is the Student's t CDF.

Compare tool: Normal vs t

In Compare mode, we overlay the standard normal and a Student’s t distribution (df = n − 1) on the same standardised axis, and show the rejection region for a chosen α and tail type.

Two-tailed critical values:
Zcrit = Φ−1(1 − α/2),    tcrit = Ft−1(1 − α/2; df)
