Purpose

The Wilcoxon signed-rank test compares paired samples using the ranks of absolute differences. It is the non-parametric alternative to the paired t-test, used when the differences cannot be assumed to be normally distributed.

When to Use

  • Paired or matched data (before vs. after, or matched pairs)
  • Differences are not assumed to be normally distributed
  • Data contains outliers or is ordinal
  • Small samples where normality cannot be verified

Key Idea

For each pair, compute the difference di = Afteri − Beforei. Rank the absolute values of the non-zero differences, then assign signs. The test asks: are the positive-rank sums and negative-rank sums balanced, or does one dominate? Zero differences are excluded from the analysis.

Hypotheses

Two-sided: H₀: Median difference = 0  |  H₁: Median difference ≠ 0

One-sided (greater): H₀: Median diff ≤ 0  |  H₁: After > Before

One-sided (less): H₀: Median diff ≥ 0  |  H₁: After < Before

Test Statistic

W⁺ = sum of ranks where di > 0 (positive ranks)

W⁻ = sum of ranks where di < 0 (negative ranks)

W = min(W⁺, W⁻) — the smaller of the two rank sums is used for the exact test.

For large samples, W is converted to a standardized z-score using a normal approximation.

Computation methods: Exact p-values enumerate all sign permutations (2n combinations). For larger n, the normal approximation is used automatically. Zero differences are always excluded.

Effect Size

Matched-pairs rank-biserial correlation (r):

r = W⁺ / (n(n+1)/2) (rescaled to −1 to +1)

  • |r| < 0.3 = small  |  0.3–0.5 = medium  |  > 0.5 = large

Assumptions

  • Observations are paired (same subjects measured twice, or matched pairs)
  • Outcome is at least ordinal — differences must be rankable
  • Pairs are independent of each other
  • Distribution of differences is symmetric (for median interpretation)

Limitations

  • Less powerful than the paired t-test when differences are truly normally distributed
  • Ties and zeros reduce test precision; many zeros weaken the test considerably
  • Does not directly produce a confidence interval for the median difference in this app

Decision Rule

Reject H₀ if p < α.

Visualisation Notes

Paired boxplots (Before / After): show the distributions of Before and After measurements side by side. Medians, quartiles, and outliers are displayed. These show your actual data — skewness is expected and valid.

Differences boxplot: shows the distribution of di = After − Before values that the test actually analyses. A red zero-line reference is shown. The location and spread of this box relative to zero is what determines the test result.

The normal curve (when shown) represents the sampling distribution of the test statistic under the null hypothesis — not your data. It only appears when using the normal approximation for large samples. The exact method does not require this curve.

Clinical Interpretation

A significant result indicates a systematic shift between the paired measurements. Report the median and IQR for Before, After, and differences alongside the p-value and effect size. Statistical significance alone does not establish clinical relevance.

Reporting Example

A Wilcoxon signed-rank test showed a significant change from before to after, W = X.XX, z = X.XX, p = 0.0X, r = X.XX. Before median = X.X (IQR: X.X–X.X); After median = X.X (IQR: X.X–X.X).

Outlier Handling

The tool flags potential outliers in the differences using the IQR rule. You can include or exclude individual pairs using the interactive data table; results update in real time. Useful for sensitivity analysis and identifying data entry errors or unusual responses.

Generating Sample Data

An expandable "Generate sample paired data" panel appears in the Data & Assumptions tab. Specify sample size, distribution (skewed/log-normal, uniform, or exponential), before median, after median, spread, and pairing correlation. Click Generate to preview and Use in analysis to load.

Tip: Higher pairing correlation (0.7–0.9) creates more realistic paired data where individual differences are consistent.
