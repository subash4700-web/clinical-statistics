PCR Tool

This tool provides an integrated workflow for sequence-focused assay planning:

  • Search by rs number (dbSNP) and fetch genomic context from NCBI
  • Load local FASTA window around the mutation position
  • Show variant details: likely consequence, population frequency, mutation type, and WT→variant nucleotide changes
  • Design basic forward/reverse flanking primers
  • Annotate restriction enzyme recognition sites on the same sequence view

Data Source

  • NCBI E-utilities esummary for SNP metadata (chromosome + position)
  • NCBI E-utilities efetch for FASTA retrieval from chromosome accession

Primer Design Notes

  • Primer3-like algorithm with nearest-neighbor thermodynamics for accurate Tm calculation
  • Evaluates self-complementarity (primer-dimer detection)
  • Checks for hairpin structures (3' end complementarity)
  • Analyzes 3' end stability and avoids long repeats
  • Generates multiple primer candidates with different positions and lengths (16-30 bp)
  • Shows quality metrics for each candidate pair (Tm, GC%, penalties, amplicon size)
  • Allows BLAST checking of individual candidates for specificity verification
  • Intended for planning/education; validate in wet-lab before use

Important

Sequence retrieval may fail if NCBI rate limits requests or if a specific rs entry lacks resolvable chromosome mapping in this MVP implementation.
