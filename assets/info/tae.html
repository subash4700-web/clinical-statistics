Purpose

This tool evaluates whether a method's combined analytical error — bias and imprecision together — falls within the clinically acceptable limit for a given analyte. It also calculates the Sigma metric, which expresses how many standard deviations of analytical error fit within the allowable error limit.

Total Error (TE)

Total error combines both sources of analytical error into a single estimate of the worst-case error expected for an individual result:

TE = |Bias%| + Z × CV%

  • Z = 1.65 — covers 95% of results (one-sided, conventional for TE)
  • |Bias%| — systematic error: the average offset from the true value
  • CV% — random error: the coefficient of variation from your precision study

Use intermediate precision CV (combining within-run and between-run variation) rather than repeatability CV alone, as TE should reflect the error a patient result actually carries under routine conditions.

Total Allowable Error (TAE)

TAE is the maximum total error permitted before a result is considered clinically unacceptable. It is set before measurement, based on:

  • Biological variation — desirable TAE = 0.5 × CVI + 1.65 × 0.5 × CVI + |allowable bias| (Ricos et al.)
  • EFLM database — use the analyte lookup in this tool for minimum, desirable, and optimum tiers
  • CLIA / regulatory — fixed proficiency testing limits for specific analytes
  • Clinical decision impact — error that would not change a clinical decision at the relevant cut-off

If TE ≤ TAE: the method is analytically acceptable. If TE > TAE: the method exceeds the allowable error budget.

Sigma Metric

The Sigma metric expresses analytical performance in units borrowed from Six Sigma manufacturing:

σ = (TAE% − |Bias%|) / CV%

It answers: how many CVs of imprecision fit between the observed bias and the allowable error limit?

QC Implications

Sigma directly informs QC design. A high-sigma method is inherently robust — errors are detected before they become clinically significant. A low-sigma method requires intensive QC to compensate for poor analytical performance.

  • σ ≥ 6: run 1 QC level with 13s rule — few false rejections, few missed errors
  • σ 4–6: standard Westgard multirule (13s, 22s, R4s, 41s)
  • σ < 4: increase QC frequency, add QC levels, investigate and correct root cause

QC cannot fix a poor-sigma method — it can only detect when it has failed. The real solution is reducing bias (recalibration, reagent change) or reducing imprecision (instrument maintenance, better reagent lots).

EFLM Performance Specifications

The EFLM lookup uses the biological variation database (Ricos et al. 2014) to provide three tiers of allowable error for each analyte:

  • Minimum — floor performance; failing this means the method is not fit for purpose
  • Desirable — recommended target for routine clinical use
  • Optimum — aspirational standard for high-complexity or critical analytes

Interpreting Results

  • TE ≤ TAE and σ ≥ 4: method is performing well — proceed with standard QC
  • TE ≤ TAE but σ = 3–4: method is borderline — monitor closely, consider bias reduction
  • TE > TAE: method fails the allowable error criterion — investigate bias and CV before clinical use
  • σ < 3: even with intensive QC, a meaningful proportion of results will exceed TAE — method improvement is needed

Always interpret Sigma in context. A σ of 3.5 for an analyte with a tight TAE (e.g. sodium) is more concerning than σ = 3.5 for an analyte where the TAE is generous. The underlying CV and bias values matter as much as the Sigma number itself.
