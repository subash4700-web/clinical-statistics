Purpose

The McNemar test evaluates whether there is a significant difference between paired proportions in a 2×2 contingency table. It is specifically designed for before–after or matched binary outcomes.

When to Use

  • Before–after studies with binary outcomes (e.g., positive/negative)
  • Matched case-control studies
  • Comparing two diagnostic tests applied to the same subjects

Key Idea

The test focuses only on discordant pairs — cases where the two measurements differ. Concordant pairs (both positive or both negative) carry no information about change and are ignored. Under H₀, discordant pairs should split roughly equally between b and c.

2×2 Paired Table

Test 2 PositiveTest 2 Negative
Test 1 Positivea — concordant (+/+)b — discordant (+/−)
Test 1 Negativec — discordant (−/+)d — concordant (−/−)

Only cells b and c (discordant pairs, shown in orange) contribute to the test. Cells a and d (concordant pairs, shown in blue) are not used.

Hypotheses

H₀: The marginal proportions are equal — b = c (no change between measurements).

H₁: The marginal proportions differ — b ≠ c (systematic change present).

Test Statistic

Without continuity correction:

χ² = (b − c)² / (b + c)

With continuity correction (Yates', recommended when b + c < 30):

χ² = (|b − c| − 1)² / (b + c)

Degrees of Freedom

df = 1

Effect Size

The odds ratio of discordant pairs describes the asymmetry of change:

OR = b / c

OR > 1: more subjects changed from negative to positive than vice versa. OR < 1: the reverse.

Alternatively, the proportion of change can be reported: (b − c) / N.

Assumptions

  • Paired or matched observations (same subjects measured twice)
  • Binary (dichotomous) outcome variable
  • Sufficient discordant pairs: b + c ≥ 10 recommended

Decision Rule

Reject H₀ if p < α.

Visualisation Notes

The 2×2 contingency table heatmap shows all four cells with colour intensity representing their values:

  • Discordant cells (b, c) — orange with red border: these are the only cells that determine the test outcome. A large difference between b and c drives significance.
  • Concordant cells (a, d) — blue: represent pairs where both measurements agreed; these are not part of the test calculation.

The test compares b (changed from positive to negative) with c (changed from negative to positive). A significant result means one direction of change dominates — indicating a treatment effect or systematic shift.

Clinical Interpretation

A significant McNemar test indicates a systematic directional change in paired binary outcomes. It is commonly used to compare two diagnostic tests on the same patients or to evaluate before–after treatment effects. Report the odds ratio and total discordant pairs (b + c) alongside the p-value.

Reporting Example

McNemar's test showed a significant difference between the two measurements, χ²(1) = X.XX, p = 0.0X. Of N discordant pairs, b = X changed from positive to negative and c = X changed from negative to positive (OR = X.XX).

Test Settings

  • Significance level (α): Choose 0.10, 0.05, or 0.01
  • Continuity correction: Yates' correction (recommended for b + c < 30)
  • Visualisation: Toggle the heatmap and chi-square distribution display
