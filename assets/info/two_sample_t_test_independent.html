Purpose

The independent two-sample t-test evaluates whether the means of two independent groups differ significantly.

When to Use

  • Comparing biomarker levels between diseased and control patients
  • Evaluating two analytical methods on different sample sets
  • Comparing treatment vs. placebo laboratory outcomes

Key Idea

Compute the difference of group means (Δ̄ = x̄₁ − x̄₂) and ask: how many standard errors is it from zero? If that distance is large relative to chance variation, H₀ is rejected. Unlike the paired test, between-subject variability cannot be removed — both within-group variance and sample sizes determine the standard error.

Hypotheses

Two-sided: H₀: μ₁ = μ₂  |  H₁: μ₁ ≠ μ₂

One-sided (greater): H₀: μ₁ ≤ μ₂  |  H₁: μ₁ > μ₂

One-sided (less): H₀: μ₁ ≥ μ₂  |  H₁: μ₁ < μ₂

Test Statistic

Welch version (default — unequal variances assumed):

t = (x̄₁ − x̄₂) / √(s₁²/n₁ + s₂²/n₂)

Pooled version (equal variances assumed):

t = (x̄₁ − x̄₂) / (sp √(1/n₁ + 1/n₂))

where sp = √[((n₁−1)s₁² + (n₂−1)s₂²) / (n₁+n₂−2)]

Degrees of Freedom

Pooled: df = n₁ + n₂ − 2

Welch: df estimated via the Welch–Satterthwaite approximation.

Confidence Intervals

95% CI of Δ̄ = (x̄₁ − x̄₂):

Δ̄ ± tdf, α/2 × SE

Answers: "Where does the true difference of means likely lie?" Narrows as n₁ + n₂ increases. This is the interval displayed in the hypothesis visualisation and tested by the independent t-test.

95% Reference range of pairwise differences (shown for illustration only — not part of the test):

Δ̄ ± tdf, α/2 × sp

Answers: "Where would individual pairwise differences typically fall if one observation were drawn from each group?" Does not narrow with larger n — individual variability is fixed. Shown in the hypothesis visualisation only to contrast with the CI above.

Effect Size

Cohen's d (standardized mean difference):

d = (x̄₁ − x̄₂) / sp

  • 0.2 = small  |  0.5 = medium  |  0.8 = large

Assumptions

  • Groups are independent (different subjects)
  • Outcome is continuous
  • Approximately normal distribution within each group (especially important for small n)
  • No extreme outliers
  • Equal variances required only for the pooled version (use Welch otherwise)

Normality Check

Normality is assessed separately for each group using the Shapiro–Wilk test and Q–Q plots.

For n ≥ 30 per group, the t-test is robust to mild non-normality due to the Central Limit Theorem. Welch's t-test is recommended as the default even when variances appear equal.

If Assumptions Are Violated

  • Use Welch's t-test if variances are unequal (it is the recommended default)
  • Consider the Mann–Whitney U test as a non-parametric alternative
  • Inspect for outliers and data-entry errors

Decision Rule

Reject H₀ if p < α.

Clinical Interpretation

A statistically significant result indicates evidence of a difference between group means. Statistical significance does not imply clinical relevance — consider the effect size (Cohen's d) and confidence interval when evaluating practical importance.

Reporting Example

An independent t-test showed a significant difference between groups, t(df) = X.XX, p = 0.0X, d = X.XX, 95% CI [L, U].
