Purpose

The chi-square test of independence assesses whether there is a statistically significant association between two categorical variables in a contingency table.

When to Use

  • Exposure vs. outcome studies (e.g., treatment vs. response)
  • Epidemiological contingency tables
  • Survey data: association between two categorical questions
  • Group comparisons on categorical outcomes

Key Idea

If two variables are independent, the distribution of one does not differ across levels of the other. Expected counts are calculated under this null assumption. The test quantifies how much the observed counts deviate from what independence predicts.

Contingency Table

For a 2×2 table, data are arranged as:

B₁B₂Row total
A₁O₁₁O₁₂R₁
A₂O₂₁O₂₂R₂
Column totalC₁C₂N

Larger tables (r × c) follow the same structure, with r rows and c columns.

Hypotheses

H₀: The two variables are independent (no association).

H₁: The two variables are associated.

Expected Counts

Under independence, expected count for each cell:

Eij = (row totali × column totalj) / N

Test Statistic

χ² = Σ (Oij − Eij)² / Eij

Degrees of Freedom

df = (r − 1)(c − 1)

where r = number of rows, c = number of columns.

Effect Size

For 2×2 tables — Phi coefficient (φ):

φ = √(χ² / N)

For larger tables — Cramér's V:

V = √(χ² / (N × min(r−1, c−1)))

  • < 0.1 = negligible  |  0.1–0.3 = small  |  0.3–0.5 = medium  |  > 0.5 = large

Assumptions

  • Observations are independent
  • Both variables are categorical
  • Expected cell counts ≥ 5 in all cells

If Assumptions Are Violated

  • If any expected count < 5, use Fisher's exact test (especially for 2×2 tables)
  • Combine sparse categories if clinically appropriate

Decision Rule

Reject H₀ if p < α.

Visualisation Notes

Chi-square distribution: shows the sampling distribution of χ² under H₀, with the rejection region (red) and the observed χ² statistic marked.

Bar chart: compares observed counts (blue) vs. expected counts (orange) for each cell in the table. Cells with large discrepancies between observed and expected are driving the association. This allows you to identify which combinations of categories deviate most from independence.

Clinical Interpretation

A significant result indicates an association between the two variables — they are not independent. The strength of association should be evaluated using φ or Cramér's V. Association does not imply causation; consider study design and potential confounders.

Reporting Example

A chi-square test of independence showed a significant association between treatment and response, χ²(1) = X.XX, p = 0.0X, Cramér's V = X.XX.
