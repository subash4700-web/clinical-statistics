Purpose

Cohen's kappa measures agreement between two raters beyond what would be expected by chance, for categorical (nominal) data.

When to Use

  • Inter-rater reliability assessment
  • Agreement between two diagnostic methods or classifiers
  • Classification consistency studies
  • Evaluating whether two observers agree beyond chance on nominal categories

Key Idea

Observed agreement (Po) alone is misleading because some agreement occurs by chance. Kappa adjusts for this: it measures how much better than chance the two raters agree. κ = 0 means agreement no better than chance; κ = 1 means perfect agreement.

Agreement Matrix

Data are arranged in a square matrix where rows = Rater 1 categories and columns = Rater 2 categories. Diagonal cells represent agreement; off-diagonal cells represent disagreement. The matrix can be any size (2×2, 3×3, etc.) depending on the number of categories.

Formula

κ = (Po − Pe) / (1 − Pe)

  • Po = observed proportion of agreement (sum of diagonal / N)
  • Pe = expected proportion of agreement by chance (based on marginal proportions)

Confidence Interval Supplementary — not part of the core Kappa test

A 95% CI for κ can be constructed using the asymptotic standard error:

SE(κ) ≈ √(Po(1 − Po) / (N(1 − Pe)²))

95% CI: κ ± 1.96 × SE(κ)

Important: The CI displayed in this tool is an asymptotic approximation that is not an official component of the Cohen's Kappa statistic. It is included as a supplementary precision indicator only. The core Kappa test produces κ, Po, and Pe — not a CI. The approximation becomes unreliable with small N (< 30) or highly imbalanced marginal distributions. Treat it as a rough guide, not a definitive interval.

Interpretation Scale

  • κ < 0.20 → Poor agreement
  • 0.21–0.40 → Fair agreement
  • 0.41–0.60 → Moderate agreement
  • 0.61–0.80 → Substantial agreement
  • 0.81–1.00 → Almost perfect agreement

These thresholds (Landis & Koch, 1977) are widely used but are arbitrary guidelines — context and clinical requirements should also guide interpretation.

Hypothesis Test

H₀: κ = 0 (agreement is no better than chance).

H₁: κ > 0 (agreement exceeds chance).

The test statistic is z = κ / SE(κ), compared to a standard normal distribution.

Assumptions

  • Exactly two raters (for standard Cohen's kappa)
  • Nominal (categorical) outcome with the same categories for both raters
  • Independent ratings — each rater classifies independently
  • All subjects rated by both raters

Limitations

  • Prevalence paradox: κ is affected by the prevalence of categories — rare categories reduce κ even with high observed agreement
  • Marginal imbalance: unequal marginal totals between raters can reduce κ
  • Does not account for the severity of disagreements (use weighted kappa for ordinal data)
  • Only applicable to exactly two raters (use Fleiss' kappa for multiple raters)

Decision Rule

Reject H₀ (κ = 0) if p < α. Focus on the κ value and its CI for practical interpretation.

Visualisation Notes

The agreement heatmap shows a matrix of observed counts for each rater combination:

  • Diagonal cells (blue): both raters agreed — the stronger the blue, the more agreements in that category
  • Off-diagonal cells (orange/red): raters disagreed — colour intensity reflects the frequency of that specific disagreement pattern

The heatmap reveals not just the overall κ value, but where disagreement occurs — which categories are most often confused between raters. This is essential for identifying systematic biases in classification.

Note on the hypothesis visualisation: The normal distribution curve and CI shown in the results are supplementary — they approximate whether κ is significantly greater than 0 using a normal SE. This is not a standard output of the Kappa test. The core result is the κ value and its verbal interpretation. Use the hypothesis display as an informal guide only.

Clinical Interpretation

Kappa is widely used to evaluate diagnostic agreement, pathology interpretation consistency, and classification reliability. Report both the κ value and its 95% CI. A statistically significant κ (> 0) is necessary but not sufficient — the magnitude must be clinically acceptable. Most clinical contexts require κ ≥ 0.6 for acceptable agreement.

Reporting Example

Agreement between raters was substantial (κ = 0.74, 95% CI: 0.62–0.86, p < 0.001).
