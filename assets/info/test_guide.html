Statistical Test Guide

The Test Guide is a step-by-step decision wizard that helps you identify the most appropriate statistical test for your data and analysis goal.

How It Works

Answer each question about your data — type, pairing, number of groups, and distribution — and the guide will recommend a specific test and let you open it directly.

Decision Branches

  • Quantitative data → t-tests, ANOVA, Mann–Whitney U, Wilcoxon
  • Qualitative / categorical data → chi-square, Fisher, McNemar, kappa, antibiotic agreement
  • Relationships → correlation, linear regression, ROC & logistic regression
  • Analytical performance → precision, bias, method comparison, reference intervals, qualitative LOD
  • Microbiology → bacterial growth, identification, colony count, enzyme activity

Analytical Performance Branch

This branch is designed around the concept of performance specifications (kravspecifikationer) — predefined allowable limits for imprecision, bias, and total error that a method must meet before clinical use. Sources include biological variation databases, EQAS/RIQAS peer data, manufacturer claims, and regulatory requirements.

  • Precision — guides between repeatability (within-run) and intermediate precision (multi-day). Enter your allowable CV% for a PASS/FAIL verdict → opens Precision Analysis
  • Trueness / Bias — systematic error against a reference. Enter your allowable bias% for a PASS/FAIL verdict → opens Bias & Trueness
  • Method comparison — guides between Bland-Altman (clinical interchangeability, limits of agreement) and regression (proportional/constant bias quantification) → opens Method Comparison
  • Reference interval — establish normal ranges from a healthy population using parametric or non-parametric methods (CLSI EP28-A3c) → opens Reference Interval
  • Qualitative LOD — determine the limit of detection for qualitative assays using probit analysis (CLSI EP12-A2) → opens Qualitative LOD

Microbiology Branch

  • Bacterial growth — plot growth phases (lag, log, stationary, death) and calculate generation time from OD or CFU data
  • Bacterial ID — systematic identification using morphology, Gram stain, and biochemical tests
  • Colony count — calculate CFU/mL from plate counts with dilution factor handling
  • Enzyme activity — calculate U/L from absorbance change, path length, and extinction coefficient
  • Antibiotic agreement — categorical agreement (EA, CA, VME, ME) between two AST methods per CLSI M52

Key Questions Asked

  • Is the goal hypothesis testing or analytical performance evaluation?
  • Is the data quantitative or qualitative?
  • How many groups are being compared?
  • Are observations paired or independent?
  • Is the distribution approximately normal?
  • What is the analysis goal (association, agreement, comparison, precision, bias)?

At the End

The guide presents the recommended test with a brief description and a direct link to open the tool. Your decision path is shown so you can trace your reasoning.

The guide provides a recommendation based on standard statistical guidance. Always consider the clinical context, sample size, and study design when selecting a test.
