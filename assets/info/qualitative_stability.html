Qualitative Stability Study

What is a qualitative stability study?

A qualitative stability study determines how long a pos/neg assay produces reliable results when samples are stored under defined conditions. The same panel of samples is tested fresh (T0) and again at each subsequent time point. Results at each time point are compared against T0 using a 2×2 contingency table.

The outcome is a functional expiry — the last time point at which the assay still meets both the Kappa acceptance criterion and the McNemar non-significance criterion.

The 2×2 table (Tx vs T0)

Rows = T0 result (reference). Columns = Tx result (stored).

  • a — T0 positive, Tx positive (concordant — stable)
  • b — T0 negative, Tx positive (overcall at Tx)
  • c — T0 positive, Tx negative (degradation — lost positive)
  • d — T0 negative, Tx negative (concordant — stable)

In most degradation scenarios you expect c > b — positive samples gradually lose reactivity as the material degrades. McNemar's test will detect this as significant directional bias.

Pass / Fail criteria

A time point PASSES when both conditions are met:

  • κ ≥ acceptance criterion (commonly 0.80 for substantial agreement)
  • McNemar p ≥ α (no statistically significant directional bias)

Functional expiry = the last consecutive passing time point. If all time points pass, expiry extends beyond the last tested point — further testing is needed.

Sample panel requirements

Include a mix of true positives and true negatives. Too few positives (a + c) makes Kappa unstable and the McNemar test insensitive. Aim for at least 30–50 samples with a realistic prevalence of positives (typically 30–50%).

Common mistake

Using percentage agreement instead of Kappa. If only 10% of samples are positive, a method that always reports negative would have 90% agreement. Kappa corrects for this — always use Kappa as the primary measure of agreement in qualitative stability studies.
