Purpose

Two-Way ANOVA examines the effect of two independent categorical factors on a continuous outcome variable, including whether the factors interact with each other.

When to Use

  • Two independent grouping variables and a continuous numeric outcome
  • Independent groups (between-subjects design)
  • Equal sample size per cell (balanced design recommended)
  • When you need to test both main effects and their interaction simultaneously

Key Idea

Each effect is evaluated using an F-ratio: F = MSEffect / MSError. Large F values indicate that between-group variation exceeds within-group variation. Two-way ANOVA tests three effects in a single model: Factor A, Factor B, and their interaction (A × B).

Design Structure

Data are arranged in cells, where each cell is one combination of Factor A and Factor B levels.

Total variance is partitioned into:

  • SSA — variation due to Factor A
  • SSB — variation due to Factor B
  • SSA×B — interaction variation
  • SSError — within-cell variation (residual)

Interaction (Interpret First)

Always interpret the interaction before the main effects.

If the interaction is significant:

  • The effect of one factor depends on the level of the other factor
  • Main effects cannot be interpreted independently
  • Interpret simple effects or inspect the interaction plot

If the interaction is NOT significant:

  • Main effects can be interpreted independently
  • Factor A and Factor B effects represent overall differences

Hypotheses

Factor A: H₀: All marginal means of A are equal  |  H₁: At least one marginal mean differs

Factor B: H₀: All marginal means of B are equal  |  H₁: At least one marginal mean differs

Interaction (A × B): H₀: No interaction between factors  |  H₁: The effect of one factor depends on the other

Test Statistic

For each effect: F = MSEffect / MSError

  • FA = MSA / MSError
  • FB = MSB / MSError
  • FA×B = MSA×B / MSError

Degrees of Freedom

  • dfA = a − 1
  • dfB = b − 1
  • dfA×B = (a − 1)(b − 1)
  • dfError = ab(n − 1)

where a = levels of Factor A, b = levels of Factor B, n = observations per cell.

Effect Size

Eta squared (η²) for each effect:

η² = SSEffect / SSTotal

  • 0.01 = small  |  0.06 = medium  |  0.14 = large

Assumptions

  • Observations are independent
  • Continuous dependent variable
  • Approximately normally distributed residuals
  • Homogeneity of variances across cells (Levene's test)
  • Balanced design preferred (equal n per cell)

If Assumptions Are Violated

  • For non-normal residuals with large n, ANOVA is generally robust
  • Unequal variances: use robust ANOVA methods or a mixed model
  • Unbalanced designs: use Type III sums of squares

Decision Rule

Reject each H₀ if p < α for that effect. Interpret interaction first, then main effects.

Clinical Interpretation

Significant main effects indicate that a factor influences the outcome overall. A significant interaction means those effects differ depending on the other factor's level — in clinical practice this often reflects that a treatment works differently across subgroups. Report effect sizes (η²) and inspect the interaction plot to communicate findings clearly.

Visual Interpretation

Marginal means plots: show sampling distributions for each level's marginal mean, scaled by that group's standard deviation. Wider curves = higher variability; narrower = lower.

Interaction plot — reading the lines:

  • Parallel lines → no interaction between factors
  • Non-parallel lines → interaction present
  • Crossing lines → strong interaction (effect reversal across levels)

Reporting Example

A two-way ANOVA revealed a significant interaction between Factor A and Factor B, F(dfA×B, dfError) = X.XX, p = 0.0X, η² = X.XX. Factor A: F(dfA, dfError) = X.XX, p = 0.0X. Factor B: F(dfB, dfError) = X.XX, p = 0.0X.

Generating Sample Data

An expandable "Generate sample data (2-way design)" panel appears below the main data entry box. Specify sample size per cell, distribution, and individual mean and SD for each Factor A × Factor B combination. Click Generate to preview the data and Use in analysis to load it.

Tip: To create an interaction, set non-parallel patterns across cells — e.g., if A1 has increasing means across B levels but A2 has decreasing means.
