Purpose

The chi-square goodness-of-fit test evaluates whether an observed categorical distribution differs significantly from a specified theoretical or expected distribution.

When to Use

  • Comparing observed frequencies to known population proportions
  • Testing fairness of a process (e.g., expected equal distribution across categories)
  • Model validation for categorical outcomes
  • Verifying expected laboratory classification distributions

Key Idea

For each category i, compute the expected count: Ei = N · pi, where N is the total sample size and pi is the expected proportion. The test quantifies how far the observed counts deviate from the expected counts across all categories.

Hypotheses

H₀: The observed distribution follows the expected (theoretical) distribution.

H₁: The observed distribution differs from the expected distribution.

Test Statistic

χ² = Σ (Oi − Ei)² / Ei

Large χ² values indicate that observed counts deviate substantially from expected counts.

Degrees of Freedom

df = k − 1

where k is the number of categories. If parameters were estimated from the data, df must be reduced accordingly.

Effect Size

Cohen's w:

w = √(χ² / N)

  • 0.10 = small  |  0.30 = medium  |  0.50 = large

Assumptions

  • Observations are independent
  • Categories are mutually exclusive and exhaustive
  • Expected proportions are specified a priori (not from the data)
  • Expected count ≥ 5 in every category

If Assumptions Are Violated

  • Combine categories to increase expected counts above 5
  • Use exact multinomial tests for small samples
  • Verify expected proportions are theoretically justified, not data-derived

Decision Rule

Reject H₀ if p < α, equivalently if χ² > χ²critical.

Visualisation Notes

Chi-square distribution: shows the sampling distribution of χ² under H₀, with the rejection region (red) and the observed χ² statistic marked. A large observed χ² falling in the red region indicates rejection of H₀.

Bar chart: side-by-side comparison of observed counts (blue) vs. expected counts (orange) for each category. Large discrepancies between blue and orange bars indicate the categories driving the significant result.

Clinical Interpretation

A significant result indicates that the observed categorical distribution deviates from the expected distribution. Statistical significance does not quantify the magnitude of deviation — report Cohen's w for practical importance.

Reporting Example

A chi-square goodness-of-fit test showed a significant deviation from the expected distribution, χ²(df) = X.XX, p = 0.0X, w = X.XX.
