Correlation & Linear Regression

Purpose

This tool quantifies the linear relationship between two continuous variables (correlation) and fits a prediction line through the data (simple linear regression). It reports Pearson and Spearman correlation coefficients, regression coefficients with confidence intervals, R², and diagnostic plots.

When to use this tool

  • Investigating whether two continuous measurements are linearly related
  • Predicting one variable from another (e.g., dose → response)
  • Quantifying the strength and direction of association
  • Reporting correlation in method-comparison or biomarker studies

Key Concepts

Pearson Correlation (r)

Measures the strength of the linear relationship between two variables. Ranges from −1 (perfect negative) to +1 (perfect positive); 0 indicates no linear association.

r = Σ[(xᵢ − x̄)(yᵢ − ȳ)] / √[Σ(xᵢ − x̄)² · Σ(yᵢ − ȳ)²]

Requires both variables to be approximately normally distributed. Sensitive to outliers.

Spearman Correlation (ρ)

A rank-based alternative to Pearson r. Robust to non-normality and outliers. Detects any monotonic (not just linear) relationship. Preferred when data are ordinal or clearly non-normal.

Confidence Interval for r (Fisher z-transformation)

The CI for Pearson r is constructed by transforming r to Fisher's z, computing a normal-theory interval, then back-transforming:

z = arctanh(r),  SE(z) = 1/√(n−3)

Reliable for n ≥ 10. Not available for Spearman (approximation only).

Simple Linear Regression

Model

ŷ = a + b · x

Estimated by Ordinary Least Squares (OLS), which minimises the sum of squared residuals.

Coefficients

  • Slope (b): Change in ŷ for a one-unit increase in x. The key inferential target.
  • Intercept (a): Predicted value of y when x = 0. May not be clinically meaningful if x = 0 is outside the data range.

Standard Errors & t-tests

Each coefficient is tested against zero using a t-statistic with df = n − 2:

t = estimate / SE

Confidence Interval for the Slope

The CI gives a range of plausible values for the true slope in the population. For example, a 95% CI of [0.65, 1.01] means: "If we repeated this study many times, 95% of the calculated intervals would contain the true slope." In practice, it tells you how precisely the slope is estimated — a narrow interval means high precision, a wide interval means uncertainty.

It is calculated as: b ± t* · SEb, where t* is the critical t-value for the chosen significance level (e.g., t* ≈ 2.0 for 95% CI with moderate sample sizes) and SEb is the standard error of the slope — smaller when you have more data or a wider spread of X values.

If the CI does not include 0, the slope is statistically significant at the chosen α.

Goodness-of-Fit

  • R²: Proportion of variance in Y explained by X. R² = r² for simple linear regression.
  • Adjusted R²: Penalises for model complexity; equals R² for simple regression but is reported for consistency.
  • RMSE: Root Mean Square Error — average deviation of observed points from the regression line, in the same units as Y.

Hypotheses

H₀: b = 0 (X does not predict Y; population slope is zero)

H₁: b ≠ 0 (X linearly predicts Y)

The p-value for the slope is equivalent to the p-value for Pearson r (both use the same t-statistic with df = n − 2).

Assumptions

  • Both variables are continuous (interval or ratio scale)
  • The relationship is linear — check the scatterplot
  • Observations are independent
  • Residuals are approximately normally distributed with constant variance (homoscedasticity)
  • No severe outliers or influential points

Diagnostic Plots

  • Scatterplot & regression line: Visually confirms linearity and identifies outliers. The confidence band shows uncertainty around the fitted line (not a prediction interval).
  • Residual plot (Residuals vs Fitted values): A residual is the difference between what the model predicted and what was actually observed: residual = observed y − predicted ŷ. The residual plot shows these differences on the Y-axis against the fitted (predicted) values on the X-axis.

    What to look for:
    • Good sign — random scatter around zero: Points are spread randomly above and below the zero line with no clear pattern. This means the model fits well and the assumptions hold.
    • Bad sign — funnel shape (spread increases): Residuals get larger as fitted values increase. This is called heteroscedasticity — the variance is not constant, which violates a key regression assumption.
    • Bad sign — curved pattern: A U-shape or arch indicates the true relationship is not linear, and a straight line is not the best fit for your data.
    • Bad sign — one or two extreme points: Large residuals far from zero are outliers that the model predicts poorly and may be worth investigating.
    Think of the residual plot as a quality check — it reveals problems that R² alone cannot show.

    Is this like a Bland-Altman plot?
    They look similar — both plot a difference on the Y-axis with a zero reference line — but they answer different questions:
    • The residual plot asks: "Is a straight line a good model for my data?" The Y-axis shows observed minus predicted (residual), and the X-axis shows the model's fitted values. It is a model diagnostic tool.
    • The Bland-Altman plot asks: "Do two measurement methods agree well enough to be used interchangeably?" The Y-axis shows Method A minus Method B, and the X-axis shows the mean of both methods. It is a clinical agreement tool.
    If you are using regression for method comparison, you would use the residual plot to validate the model, then interpret the slope and intercept clinically. However, Bland-Altman is generally preferred for pure method comparison because it focuses on clinically meaningful agreement — not model fit.

Important Cautions

  • Correlation ≠ causation. A significant r or slope does not imply that X causes Y.
  • Extrapolation: Predictions outside the observed range of X are unreliable.
  • Outliers: A single influential point can substantially change r and the slope.
  • Restriction of range: Correlation underestimates association if the data cover only a narrow slice of the true range.

Reporting Example

There was a strong positive linear relationship between systolic blood pressure and age (r = 0.72, 95% CI [0.61, 0.81], p < 0.001). Simple linear regression showed that each additional year of age was associated with an increase of 0.83 mmHg in systolic blood pressure (b = 0.83, SE = 0.09, 95% CI [0.65, 1.01], t(98) = 9.2, p < 0.001, R² = 0.52).

Tool Features

Study Context

Verify assumptions before running the analysis:

  • Both variables are continuous
  • Linear relationship expected (verify with scatterplot)
  • Observations are independent
  • Approximately normal (required for Pearson; Spearman is robust)

Settings

  • Significance level (α): 0.05 or 0.01
  • Confidence band: Toggle display of the CI band around the regression line
  • Residual plot: Toggle the residuals vs fitted values diagnostic plot

Data Input

Enter X and Y values one per line (or comma/semicolon/tab-separated). Rows are paired — X[1] is matched with Y[1]. Both columns must have equal length and at least 3 pairs.

Sample Data Generator

Generates correlated normal data with a target Pearson r:

  • n: Sample size
  • Target r: Desired correlation (−0.999 to 0.999)
  • X mean / SD: Parameters for the X distribution
