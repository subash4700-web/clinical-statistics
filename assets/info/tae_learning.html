Understanding Total Error & Sigma

What is total error?

Total error (TE) combines both sources of analytical error — imprecision (random scatter) and bias (systematic offset) — into a single estimate of how far a patient result might be from the true value in the worst case.

TE% ≈ |Bias%| + 1.65 × CV%

The factor 1.65 captures the 95th percentile of random error (one-tailed). Some labs use 2× CV% for a more conservative estimate.

Allowable total error (TEa)

TEa is the maximum total error a method is allowed to have before it affects clinical decisions. It is set by biological variation (EFLM/Ricos) or regulatory bodies (e.g. CLIA). If TE < TEa, the method meets the performance goal.

The Sigma metric

The sigma metric tells you how many standard deviations of imprecision fit between zero bias and the allowable error limit:

σ = (TEa − |Bias%|) / CV%

  • σ ≥ 6 — world-class performance. Minimal QC needed.
  • σ 4–6 — good performance. Standard Westgard rules sufficient.
  • σ 3–4 — marginal. Tighter QC rules required.
  • σ < 3 — poor. Method likely needs improvement before clinical use.

Why it matters

Sigma directly determines your QC strategy. A high-sigma method rarely produces false rejections and rarely misses errors — you can run fewer QC samples. A low-sigma method requires more frequent QC and tighter rules to catch errors before they reach patients.

Common mistake

Calculating sigma without checking which TEa to use. EFLM biological variation goals, CLIA limits, and manufacturer claims are all different targets. Always state which TEa you used and why. Desirable biological variation goals are the most common starting point.
