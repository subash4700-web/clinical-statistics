Purpose

The paired t-test compares two dependent measurements taken on the same subjects, testing whether the mean within-subject difference differs from zero.

When to Use

  • Before–after designs (same subjects measured twice)
  • Repeated measurements on the same subjects
  • Matched pairs (e.g., left/right measurements)

Key Idea

For each subject i, compute a difference:

di = Afteri − Beforei

The test then reduces to a one-sample t-test on these differences, asking whether their mean (d̄) differs from zero. By working on differences, between-subject variability is removed — making the test more powerful than an independent t-test when pairing is valid.

Hypotheses

Two-sided: H₀: μd = 0  |  H₁: μd ≠ 0

One-sided (greater): H₀: μd ≤ 0  |  H₁: μd > 0

One-sided (less): H₀: μd ≥ 0  |  H₁: μd < 0

Test Statistic

t = d̄ / SE   where   SE = sd / √n

  • d̄ = mean of the differences
  • sd = SD of the differences
  • n = number of pairs

Degrees of Freedom

df = n − 1

Confidence Intervals

95% CI of the mean difference:

d̄ ± tn−1, α/2 × SE   where   SE = sd / √n

Answers: "Where does the true mean difference likely lie?" Narrows as n increases. This is the interval displayed in the hypothesis visualisation and tested by the paired t-test.

95% Reference range of individual differences (shown for illustration only — not part of the test):

d̄ ± tn−1, α/2 × sd

Answers: "Where do 95% of individual subject differences fall?" Does not narrow with larger n — individual differences always scatter around the mean. Shown in the hypothesis visualisation only to contrast with the CI above.

Effect Size

Cohen's dz (standardized mean difference on differences):

dz = d̄ / sd

  • 0.2 = small  |  0.5 = medium  |  0.8 = large

Assumptions

  • Measurements are paired (dependent samples)
  • Outcome is continuous (numeric measurement)
  • Pairs are independent of each other
  • Differences are approximately normally distributed
  • No extreme outliers in the differences (important for small n)

Normality Check

Normality is assessed on the differences, not on the raw before/after values. Use the Shapiro–Wilk test and Q–Q plot on di values.

For n ≥ 30, the t-test is robust to mild non-normality due to the Central Limit Theorem.

If Normality Is Violated

  • Consider the Wilcoxon signed-rank test as a non-parametric alternative
  • Inspect for outliers in the differences

Decision Rule

Reject H₀ if p < α.

Clinical Interpretation

A statistically significant result suggests a systematic change between the paired measurements. Statistical significance does not automatically imply clinical relevance. Consider the magnitude of change (effect size dz) and the confidence interval.

Reporting Example

A paired t-test showed a significant change from before to after, t(df) = X.XX, p = 0.0X, dz = X.XX, 95% CI [L, U].
