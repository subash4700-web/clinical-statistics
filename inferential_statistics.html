1. What is inferential statistics?

When we measure something in a lab or a clinical study, we never have access to the entire population — we work with a sample. Inferential statistics gives us a principled way to draw conclusions about a population based on that sample, and to quantify how much uncertainty comes with those conclusions.

The core question is always: Could the pattern we see in our sample have occurred by chance alone, or does it reflect something real in the population?

2. Hypothesis testing

Every hypothesis test starts with two competing statements:

Null hypothesis (H₀): There is no effect, no difference, or no association. Any observed difference is due to random sampling variation.

Alternative hypothesis (H₁): There is a real effect or difference in the population.

We never "prove" H₀ or H₁. Instead, we ask: assuming H₀ is true, how likely is it to observe data at least as extreme as what we got? That probability is the p-value.

3. The p-value

The p-value is the probability of obtaining a result as extreme as — or more extreme than — the one observed, assuming the null hypothesis is true.

Correct interpretation: A p-value of 0.03 means: if H₀ were true, there would be a 3% chance of seeing a difference this large (or larger) purely by chance.
Common misconceptions:
✗ "p = 0.03 means there is a 3% chance H₀ is true." — Wrong.
✗ "p = 0.03 means there is a 97% chance the result is real." — Wrong.
✗ "A smaller p-value means a bigger effect." — Wrong.

In practice, the p-value is treated as dichotomous: either p < α (reject H₀, "significant") or p ≥ α (fail to reject H₀, "not significant"). A p = 0.049 and p = 0.051 represent essentially the same evidence, yet are treated completely differently by this threshold.

4. Significance level (α)

The significance level α is the threshold we set before the experiment. It defines how willing we are to make a Type I error — falsely rejecting a true null hypothesis (a false positive).

The most commonly used threshold is α = 0.05, meaning we accept a 5% risk of a false positive. This is a convention, not a law — in some contexts (e.g. genomics, safety-critical decisions) much lower α values are used.

Key point: α is set before data collection. Changing it after seeing the results is a form of p-hacking and invalidates the test.

5. Type I and Type II errors

Every decision in hypothesis testing carries two possible error types:

H₀ is true (no real effect)H₀ is false (real effect exists)
Reject H₀ (p < α)Type I error — false positive (α)Correct decision — Power (1 − β)
Fail to reject H₀ (p ≥ α)Correct decision (1 − α)Type II error — false negative (β)
Type I error (α): You conclude there is an effect when there isn't one. Setting α = 0.05 means you accept a 5% risk of this.

Type II error (β): You fail to detect a real effect. Conventionally β = 0.20, meaning a 20% chance of missing a true effect.

There is an inherent trade-off: lowering α reduces false positives but increases false negatives, and vice versa. The only way to reduce both simultaneously is to increase sample size.

6. Statistical power

Power is the probability of correctly detecting a true effect — rejecting H₀ when H₁ is actually true. Power = 1 − β. A study with 80% power has a 20% chance of missing a real effect.

Power is determined by three factors:

FactorEffect on power
Sample size (n)Larger n → higher power. More data makes it easier to detect a true difference.
Effect size (Δ)Larger true difference → higher power. A large signal is easier to detect.
Significance level (α)Higher α → higher power, but at the cost of more false positives.
Try it: Open the Dance of the p-values simulator. Increase n while keeping the effect fixed — watch power rise and significant results become more frequent.
Underpowered studies: A non-significant result in an underpowered study does not mean there is no effect. It may simply mean the study was too small to detect it.

7. Confidence intervals

A 95% confidence interval (CI) gives a range of plausible values for the population parameter based on the sample data.

Correct interpretation: If you repeated the same experiment 100 times and computed a 95% CI each time, approximately 95 of those intervals would contain the true population value. Any single CI either contains the true value or it doesn't — we just don't know which.
Common misconception: "There is a 95% probability that the true value lies within this specific interval." — The true value is fixed; it is the interval that varies between experiments.

The width of the CI reflects precision: a narrow CI means more precise estimation. Increasing sample size narrows the CI. You can observe this directly in the Dance of the p-values — as n increases, the forest of CIs becomes noticeably tighter.

8. Statistical vs. clinical significance

A statistically significant result (p < α) tells you only that the observed difference is unlikely to be due to chance. It says nothing about whether that difference is clinically meaningful.

With a large enough sample size, almost any difference — no matter how trivially small — will eventually become statistically significant. This is why effect size matters.

MeasureWhat it tells you
p-valueWhether the result is likely due to chance
Effect size (e.g. Cohen's δ)How large the difference is in standardised units, independent of sample size
Clinical significanceWhether the difference matters in practice — a clinical judgement, not a statistical one
Cohen's δ guidelines (approximate):
0.2 = small  ·  0.5 = medium  ·  0.8 = large

A δ of 0.05 with p < 0.001 may be statistically robust but clinically irrelevant. A δ of 0.8 with p = 0.08 may be clinically important but the study was underpowered to detect it.
Always report effect size alongside the p-value. Statistical significance without effect size is incomplete reporting.

9. Categorical tests

Categorical tests are used when the outcome variable is not a continuous measurement but a category — for example: positive/negative, disease present/absent, or agreement/disagreement between two raters.

Like all inferential tests, they produce a p-value that answers: if H₀ is true, how likely is this distribution of counts by chance?

TestUse caseH₀
Chi-Square GOF Do observed frequencies match an expected distribution? Observed proportions equal expected proportions
Chi-Square Independence Are two categorical variables associated? The two variables are independent
Fisher's Exact Test Same as Chi-Square Independence, but for small samples where expected cell counts are low (<5) The two variables are independent
McNemar's Test Change in paired categorical data (e.g. before/after, two raters on same subjects) No change in proportions between paired measurements
Kappa / Weighted Kappa Agreement between two raters or methods beyond chance Agreement is no better than chance (κ = 0)
When to use Fisher's Exact vs Chi-Square: Use Fisher's Exact when any expected cell count is < 5. For larger samples, Chi-Square is appropriate and computationally simpler.
Kappa note: Kappa measures agreement beyond what is expected by chance. A κ of 1.0 = perfect agreement; κ = 0 = agreement no better than chance; κ < 0 = systematic disagreement. It is more of a descriptive measure than a pure hypothesis test, though a p-value can be computed.

All categorical tests assume that observations are independent (each subject contributes only one observation) — except McNemar, which is explicitly designed for paired data.

αYour accepted false positive rate (typically 0.05), set before the experiment
