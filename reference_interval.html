CLSI EP28-A3c — Establishing and verifying reference intervals

How to use: Paste or enter reference values from healthy individuals (one value per line or comma-separated). Minimum 120 samples recommended (EP28). The tool calculates the 2.5th and 97.5th percentiles with 90% confidence intervals. Outliers are detected using Tukey's method (1.5 × IQR) or Dixon's Q test.

EP28-A3c Guidelines:
Sample size: Minimum 120 reference individuals recommended. With fewer samples, confidence intervals widen significantly.
Non-parametric method: Recommended when distribution is unknown. Uses ranked values (2.5th and 97.5th percentiles).
Parametric method: Use only when data is normally distributed. Calculates mean ± 1.96 × SD.
Outlier removal: Outliers should be investigated clinically before removal. Maximum 1/3 of data should be removed.
Partitioning: Consider sex, age, or other subgroups if clinically relevant (Harris & Boyd method).
