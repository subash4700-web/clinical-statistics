📈 Understanding Method Comparison

Simulate paired measurements — explore bias types, Bland-Altman analysis, and the correlation trap.

Exercise 1
Methods agree — ideal result
Both methods return similar values with good precision. Scatter clusters along the identity line. Bland-Altman is flat around zero.
Slope = 1.00 Intercept = 0 CV = 2%
Exercise 2
Constant bias — fixed offset
Method B reads consistently higher by a fixed amount at all concentrations. r is still near 1.0 — but the methods are NOT interchangeable.
Slope = 1.00 Intercept = +8 CV = 2%
Exercise 3
Proportional bias — fanning
Method B reads ~10% higher and the gap grows with concentration. Bland-Altman differences slope upward — they are not flat.
Slope = 1.10 Intercept = 0 CV = 2%
Exercise 4
The r trap — high r, real bias
Pearson r looks excellent (~0.99). But there is a large constant bias. Correlation measures association, not agreement. Always use Bland-Altman.
r ≈ 0.99 Intercept = +15 Range 5–60
Exercise 5
Small n — CI too wide to conclude
Real proportional bias exists (slope = 1.12) but with only 20 samples the 95% CI of the slope still includes 1.0. Statistically inconclusive — not the same as no bias.
n = 20 Slope = 1.12 Wide CI band
Exercise 6
Confirmed proportional bias
Same 12% proportional bias but with n = 80. The CI of the slope now excludes 1.0 — bias is statistically confirmed. The CI band on the scatter plot no longer overlaps the identity line.
n = 80 Slope = 1.12 CI excludes 1.0
