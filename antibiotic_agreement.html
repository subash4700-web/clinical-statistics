Interpretation Guide — How to Read the 3x3 Matrix

The 3x3 matrix compares the reference method (gold standard) against the test method (the method being evaluated). Each cell shows how many isolates fell into that combination. The diagonal cells (S/S, I/I, R/R) are agreements. Off-diagonal cells are errors — classified by clinical severity:

Reference ↓ / Test → Susceptible (S) Intermediate (I) Resistant (R)
Susceptible (S) ✅ Agreement Minor Error (mE) Major Error (ME)
Intermediate (I) Minor Error (mE) ✅ Agreement Minor Error (mE)
Resistant (R) Very Major Error (VME) Minor Error (mE) ✅ Agreement
🔴
Very Major Error (VME) — Reference = R, Test = S
The organism is truly resistant, but the test method calls it susceptible. The patient receives an antibiotic that will not work. This is the most dangerous error — it can lead to treatment failure, disease progression, and increased mortality.
Acceptance limit: < 3% of resistant isolates
🟠
Major Error (ME) — Reference = S, Test = R
The organism is truly susceptible, but the test method calls it resistant. The patient is unnecessarily switched to a broader-spectrum antibiotic. This contributes to antimicrobial resistance and may increase side effects and cost.
Acceptance limit: < 3% of susceptible isolates
🟡
Minor Error (mE) — One method = I, the other = S or R
A disagreement involving the intermediate category. Less clinically consequential because the I zone is a buffer — it signals that the antibiotic may work at higher doses or specific infection sites. Dose adjustment rather than drug change is typically considered.
Preferred limit: < 10% of all isolates
🟢
Agreement — Both methods report the same category (S/S, I/I, or R/R)
The test method correctly reproduces the reference classification. The diagonal of the 3x3 matrix represents all agreements.
Target: Category Agreement (CA) ≥ 90%

Definitions (EUCAST):
Category Agreement (CA): Same S/I/R category between test and reference. Target: >= 90%.
Very Major Error (VME): Reference = R, Test = S. False susceptible — most dangerous. Target: < 3%.
Major Error (ME): Reference = S, Test = R. False resistant — may deny effective treatment. Target: < 3%.
Minor Error (mE): One method = I, other = S or R. Target: no formal limit, but < 10% preferred.
Essential Agreement (EA): MIC within ±1 doubling dilution. Not applicable for disc diffusion — only for MIC methods.
